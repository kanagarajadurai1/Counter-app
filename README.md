# ⚡ Counter App

A simple, clean Counter Application built with **React** and **useState** hook.

## 🚀 Live Demo
[View Live](https://your-vercel-link.vercel.app)

## 🛠️ Built With
- React.js
- useState Hook
- CSS-in-JS

## ✨ Features
- ➕ Increment counter by 1
- ➖ Decrement counter by 1
- 🔄 Reset counter to zero
- 🚫 Prevents counter from going below zero
- ⚠️ Shows "Minimum limit reached" message at zero

## 📦 Getting Started
```bash
git clone https://github.com/kanagarajadurai1/Counter-app.git
cd Counter-app
npm install
npm start
```

## 🌐 Deployment
Deployed on **Vercel** — auto-deploys on every push to main.
